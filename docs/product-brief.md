# 제품 개요

## 제품 한 줄 정의

독일어 중심 UI와 야구 용어 기반 카피로 팀 수비 상황을 학습하는 모바일 퍼스트 인터랙티브 학습 앱.

## 문제 정의

취미 야구 팀에서는 수비 포지셔닝, 커버, 백업, 송구 우선순위를 말로만 설명하면 이해가 느립니다.
특히 초보자나 새 팀원은 "내가 지금 어디로 움직여야 하는지"를 상황별로 연결해서 익히기 어렵습니다.

## 해결 방식

앱은 특정 장면을 선택하면 아래 정보를 한 화면 흐름으로 연결합니다.

- 전체 수비 정렬
- 공 처리자와 빈 베이스 커버
- 백업과 다음 플레이 준비
- 포지션별 첫 스텝과 책임
- 독일어 설명
- 영어 야구 용어를 필요한 곳에서 유지하는 혼합 카피
- 긴 세로 스크롤보다 `Ablauf -> Position` 모드 전환
- 카테고리에서 상세로 들어간 직후 핵심 애니메이션 필드를 먼저 보여주는 포커스 이동

## 핵심 사용자

- 독일에서 취미로 야구하는 선수
- 팀 내 초보자 또는 새로 포지션을 바꾼 선수
- 팀 수비를 체계적으로 배우고 싶은 사용자

## 핵심 가치

- 복잡한 수비 판단을 시각적으로 빠르게 이해할 수 있음
- 한 포지션에 집중해서 자기 역할만 깊게 볼 수 있음
- 필드를 계속 보면서 장면 이해와 포지션 이해를 자연스럽게 오갈 수 있음

## 현재 범위와 목표

- 현재 코드 기준 구현 시나리오: 11개
- 6개 구현 카테고리 노출
- 문서 정본 기준 1차 커리큘럼: 36개
- 확장 단위: `카테고리 -> 시나리오 -> 포지션 학습`
- 사용자 노출 언어: 독일어 기본, 핵심 야구 용어는 영어 허용
- 문서 원문 언어: 한국어, 독일어 문구는 구현 단계에서 파생

## MVP 기능

- 독일어 중심 UI
- 카테고리 기반 시나리오 선택
- 홈의 6개 카테고리 텍스트 카드 보드
- 필드 애니메이션
- 상세 진입 시 애니메이션 포커스 자동 이동
- 상세 화면의 `Ablauf / Position` 모드 전환
- 포지션 포커스 패널
- 카테고리 화면으로 돌아가 다음 시나리오를 고르는 반복 학습 흐름

## MVP에 넣지 않을 것

- 회원가입
- 팀별 계정/권한
- 영상 업로드
- 복잡한 기록 분석
- 초기 퀴즈 모드
- 초기 LLM Coach
- 초기부터 36개 전부를 한 번에 구현하는 방대한 기능 확장
- 전용 백엔드 서버 구축

## 사용자 흐름

1. 사용자가 카테고리를 고른다.
2. 사용자가 카테고리 안에서 시나리오를 고른다.
3. 앱이 상세 화면으로 들어가자마자 애니메이션 필드가 먼저 보이도록 포커스를 맞춘다.
4. 앱이 `Ablauf` 화면에서 9명의 움직임과 공의 흐름을 애니메이션으로 보여준다.
5. 사용자가 `Position` 화면으로 전환해 특정 포지션의 책임과 송구 선택을 본다.
6. 사용자는 카테고리 화면으로 돌아가 다음 시나리오를 반복 학습한다.

## 성공 기준

- 사용자가 특정 상황에서 자기 포지션의 이동 목적을 설명할 수 있음
- 사용자가 `Ablauf`를 보고 팀 콜과 Primary Out, Fallback을 설명할 수 있음
- 사용자가 `Position`에서 자기 역할과 커버/백업을 빠르게 찾을 수 있음
- 사용자가 카테고리에서 상세로 들어간 직후 스크롤 없이 핵심 애니메이션을 볼 수 있음
- 팀 공통 수비 언어를 배우는 학습 도구로 사용할 수 있음
