# 03. 번트/스퀴즈 레이아웃

이 문서는 `번트/스퀴즈` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [03-bunts-and-squeezes.md](./03-bunts-and-squeezes.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 번트 카테고리는 `corners-in 시작점`, `P 1루 커버`, `middle rotation`, `코너 베이스 백업`을 좌표로 고정합니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지합니다.

## 카테고리 공통 규칙

- 3루 쪽 번트의 `3B` 시작은 `(33.50, 64.16)` 계열을 기본으로 합니다.
- 1루 쪽 번트 대응의 `1B` 시작은 `(67.50, 64.75)` 계열을 기본으로 합니다.
- `P`의 1루 커버 최종점은 `(69.00, 63.00)`을 사용합니다.
- squeeze 계열에서는 `LF`가 `3B line backup`, `RF`가 `1B foul backup`, `CF`가 `2B deep backup`을 맡습니다.
- `BALL`의 번트 구간은 공을 잡기 전까지 항상 해당 파울라인 `안쪽 fair territory`를 유지합니다.
