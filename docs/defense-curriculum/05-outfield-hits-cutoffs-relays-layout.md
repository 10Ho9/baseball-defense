# 05. 외야 안타/컷오프/중계 레이아웃

이 문서는 `외야 안타/컷오프/중계` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [05-outfield-hits-cutoffs-relays.md](./05-outfield-hits-cutoffs-relays.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 이 카테고리는 `처리 외야수 포구점`, `cutoff 깊이`, `stop-base`, `반대편 외야 백업`을 좌표로 고정합니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지합니다.

## 카테고리 공통 규칙

- 처리 외야수의 첫 field point는 타구 방향 쪽으로 `y 40.00~47.00` 구간에서 잡습니다.
- Home cutoff는 보통 `x 41.00~59.00`, `y 54.00~56.00` 구간에서 정렬합니다.
- 3루 stop-base는 `(31.00, 63.00)`을 사용합니다.
- 반대편 코너 외야수는 반드시 가운데로 안쪽 이동해 relay miss의 2차 백업이 됩니다.
- 처리 외야수의 `Move 2 / Hold`가 릴리스 지점이면, 얕은 안타/캐치 장면에서는 `Move 1`보다 송구 방향으로 이어지게 두고 더 깊은 외야 쪽으로 물리지 않습니다.
- `23~25`의 Home 관련 장면에서도 처리하지 않는 코너 외야수는 홈 라인까지 내려오지 않고, `y 31.00~38.00` 계열의 중간 리바운드 백업에 머뭅니다.
