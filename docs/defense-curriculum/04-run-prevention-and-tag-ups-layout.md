# 04. 득점 저지/태그업 레이아웃

이 문서는 `득점 저지/태그업` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [04-run-prevention-and-tag-ups.md](./04-run-prevention-and-tag-ups.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 이 카테고리는 `Home receive`, `플레이트 뒤 백업`, `stop-base`, `no-play 전환`의 좌표를 고정합니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지합니다.

## 카테고리 공통 규칙

- Home receive 기준점은 `(50.00, 86.50)`입니다.
- `P`의 플레이트 뒤 백업 최종점은 `(50.00, 90.00)` 계열을 사용합니다.
- `LF`와 `RF`가 `(39.00, 84.00)` / `(61.00, 84.00)` 계열의 짧은 홈 라인 백업을 맡는 것은 `19`, `20`처럼 타구가 내야-플레이트 근처에 있는 장면에 한정합니다.
- `21`, `22`처럼 외야 캐치 후 Home 송구를 만드는 장면에서는 코너 외야수가 플레이트까지 내려오지 않고, cutoff 뒤 `y 31.00~38.00` 계열의 중간 리바운드 백업에 머뭅니다.
- 태그업 송구가 애매하면 cutoff는 `y 54.00~56.00` 구간에서 no-play 전환을 준비합니다.
