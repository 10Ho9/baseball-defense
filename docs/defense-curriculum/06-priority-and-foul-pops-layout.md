# 06. 플라이 우선권/파울팝 레이아웃

이 문서는 `플라이 우선권/파울팝` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [06-priority-and-foul-pops.md](./06-priority-and-foul-pops.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 이 카테고리는 `우선권을 가진 선수의 진입 라인`, `양보 선수의 peel-off`, `2루 커버`, `깊은 안전판`을 좌표로 고정합니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지합니다.

## 카테고리 공통 규칙

- 우선권을 가진 선수는 공 아래로 `직선 진입`하고, 양보 선수는 충돌을 피하기 위해 바깥쪽 호로 빠집니다.
- 짧은 플라이는 `2B` 또는 `SS`가 `(50.50, 44.50)` 계열을 기준으로 2루 전환을 준비합니다.
- 파울팝은 이 문서에서 `1루 쪽은 1B 우선`, `3루 쪽은 3B 우선`으로 고정합니다.
