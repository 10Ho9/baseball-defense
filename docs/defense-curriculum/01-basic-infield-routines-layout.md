# 01. 기본 내야 루틴 레이아웃

이 문서는 `기본 내야 루틴` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [01-basic-infield-routines.md](./01-basic-infield-routines.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 여기 적힌 값은 `positions.start`, `positions.path`, `backups`, `actorTracks`의 원본 좌표입니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지하고, 해당 카테고리의 기본 깊이를 그대로 사용합니다.

## 카테고리 공통 규칙

- 1루 마무리 좌표는 기본적으로 `(69.00, 63.00)`을 사용합니다.
- 1루선 오버스로우 백업은 `P` 또는 `RF`가 `(82.00, 68.00)` 계열로 닫습니다.
- 좌측 루틴에서는 `LF`가 `(18.00~20.00, 44.50~47.50)` 구간으로 들어와 첫 백업을 맡습니다.
- 우측 루틴에서는 `RF`가 `(80.00~82.50, 44.50~68.00)` 구간으로 들어와 첫 백업을 맡습니다.
- `CF`는 코너 외야가 전진하면 비는 갭을 닫기 위해 `Left-center pinch` 또는 `Right-center pinch`로 이동합니다.
- 애니메이션 구현은 항상 `타구 접촉 -> 포구 지점 -> 발정렬/transfer -> 릴리스 또는 자가 처리 -> 1루 마무리` 순서를 먼저 고정합니다.
- 표의 `Move 1`은 기본적으로 `포구 지점`, `Move 2 / Hold`는 `송구 발정렬`, `커버 확정`, `자가 처리 마무리` 중 하나로 해석합니다.
- `throwEvents` 종료 시점과 `BALL`의 1루 도착 시점은 사실상 같아야 합니다.
