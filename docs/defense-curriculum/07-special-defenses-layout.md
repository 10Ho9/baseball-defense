# 07. 특수 수비 레이아웃

이 문서는 `특수 수비` 카테고리의 좌표 원본문서입니다.

- 전술 판단 원문: [07-special-defenses.md](./07-special-defenses.md)
- 공통 좌표 기준: [field-layout-reference.md](../field-layout-reference.md)
- 이 카테고리는 `도루 커버`, `1·3루 cut/hold`, `브레이크 플레이`, `런다운 교환 라인`을 좌표로 고정합니다.
- 표에 없는 포지션은 기본 시작 좌표를 유지합니다.

## 카테고리 공통 규칙

- 도루 커버는 이 문서에서 `SS가 2루 커버`하는 버전으로 고정합니다.
- 1·3루 상황에서 `CF`는 얕은 정중앙으로 들어와 2루 miss와 Home return miss를 동시에 정리합니다.
- 런다운은 `공 가진 선수는 끝까지 쫓지 않고 한 번 더 끌어낸 뒤 교환`하는 동선을 좌표로 표현합니다.
